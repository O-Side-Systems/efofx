**Extend `app/services/auth_service.py`** — Add these functions (keep register/verify from 02-01):

1. `create_access_token(tenant_id: str, user_id: str, role: str) -> str`:
   - Payload: `{"sub": user_id, "tenant_id": tenant_id, "role": role, "iat": now, "exp": now + 20 minutes}`
   - Use `jwt.encode(payload, settings.JWT_SECRET_KEY, algorithm="HS256")` — PyJWT 2.x returns str directly, no .decode()
   - Use `datetime.now(timezone.utc)` for all timestamps (NOT `datetime.utcnow()`)

2. `create_refresh_token() -> tuple[str, str]`:
   - `raw = secrets.token_urlsafe(48)` for raw token
   - `hashed = password_hash.hash(raw)` for storage
   - Return `(raw, hashed)`

3. `login_tenant(body: LoginRequest) -> LoginResponse`:
   - Fetch tenant by email from tenants collection
   - If not found: raise 401 "Invalid credentials" (generic — per locked decision, never reveal email existence)
   - If found but not active: raise 401 "Invalid credentials"
   - Verify password with `password_hash.verify(body.password, tenant["hashed_password"])`
   - If password wrong: raise 401 "Invalid credentials" (same message as not found)
   - If not email_verified: raise 403 "Email not verified. Check your inbox."
   - Generate access_token via create_access_token(tenant_id, tenant_id, "owner") — user_id = tenant_id for single-user tenants
   - Generate refresh_token via create_refresh_token()
   - Store refresh token hash in refresh_tokens collection: `{"hashed_token": hashed, "tenant_id": tenant_id, "expires_at": now + 14 days, "created_at": now}`
   - Return LoginResponse with access_token, raw refresh_token, "bearer", and expires_in=1200 (20 min in seconds)

4. `refresh_access_token(body: RefreshRequest) -> TokenResponse`:
   - Fetch all refresh tokens for lookup (or use a smarter approach: iterate recent tokens)
   - Better approach: Store a `token_hash_prefix` (first 12 chars of the hash) as an indexed field. Look up by prefix, then verify full hash.
   - Actually, simplest secure approach: Store the refresh token as a hash. On refresh, we need to find it. Use `secrets.token_urlsafe(48)` and store a SHA-256 hash (not bcrypt) of the raw token for O(1) lookup. Bcrypt is one-way and unsearchable.
   - **Revised approach:** Use `hashlib.sha256(raw.encode()).hexdigest()` as the lookup key in MongoDB. This is safe because refresh tokens have 384 bits of entropy (token_urlsafe(48)).
   - Look up by SHA-256 hash in refresh_tokens collection
   - If not found or expired: raise 401 "Invalid refresh token"
   - Delete the used token (rotation — one-time use)
   - Create new access_token and new refresh_token
   - Store new refresh token hash
   - Return TokenResponse with new access_token and expires_in=1200

  ```python
  async def get_current_tenant(
      credentials: HTTPAuthorizationCredentials = Depends(security)
  ) -> Tenant:
      if not credentials:
          raise HTTPException(status_code=401, detail="Authentication required")

      token = credentials.credentials

      # Check if API key auth (starts with "sk_live_")
      if token.startswith("sk_live_"):
          return await _validate_api_key(token)

      # JWT auth
      try:
          payload = jwt.decode(
              token,
              settings.JWT_SECRET_KEY,
              algorithms=["HS256"],
              options={"require": ["exp", "iat", "sub", "tenant_id", "role"]},
          )
      except jwt.ExpiredSignatureError:
          raise HTTPException(status_code=401, detail="Token expired")
      except jwt.InvalidTokenError:
          raise HTTPException(status_code=401, detail="Invalid token")

      db = get_database()
      tenant_doc = await db[DB_COLLECTIONS["TENANTS"]].find_one(
          {"tenant_id": payload["tenant_id"]}
      )
      if not tenant_doc:
          raise HTTPException(status_code=401, detail="Invalid token")

      if not tenant_doc.get("email_verified"):
          raise HTTPException(status_code=403, detail="Email not verified")

      return Tenant(**tenant_doc)
  ```
- Add `_validate_api_key(raw_key: str) -> Tenant` helper:
  - Extract tenant_id from key format `sk_live_{tenant_id_no_dashes}_{random}` — parse first 32 chars after `sk_live_` as tenant_id (re-insert UUID dashes)
  - Fetch tenant by tenant_id
  - Verify hash: `password_hash.verify(raw_key, tenant["hashed_api_key"])`
  - If invalid: raise 401
  - If not email_verified: raise 403
  - Return Tenant
