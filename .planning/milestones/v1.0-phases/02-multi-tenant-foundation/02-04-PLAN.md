<objective>
Implement BYOK (Bring Your Own Key) encryption for OpenAI API keys using per-tenant Fernet keys derived via HKDF, with key storage, rotation, and validation endpoints.

Purpose: Contractors must store their own OpenAI API key to use LLM features. This plan ensures keys are encrypted at rest using per-tenant derived keys (limiting blast radius), validated against OpenAI before storage, and decryptable per-request without ever persisting plaintext. Per the locked decision, there is no platform key fallback — LLM features are blocked until a key is stored.

Output: Crypto utilities, BYOK service layer, key management endpoint, and the 402 gate for missing keys. TDD approach because encryption has well-defined I/O with zero tolerance for errors.
</objective>

**GREEN phase — Create `app/utils/crypto.py`:**

Follow the pattern from 02-RESEARCH.md Pattern 2 exactly:

```python
import base64
from cryptography.fernet import Fernet
from cryptography.hazmat.primitives.kdf.hkdf import HKDF
from cryptography.hazmat.primitives import hashes


def derive_tenant_fernet_key(master_key: bytes, tenant_id: str) -> Fernet:
    """Derive a unique Fernet key for a tenant using HKDF."""
    hkdf = HKDF(
        algorithm=hashes.SHA256(),
        length=32,
        salt=None,  # master_key is high-entropy
        info=f"efofx-byok-{tenant_id}".encode(),
    )
    derived_key = hkdf.derive(master_key)
    fernet_key = base64.urlsafe_b64encode(derived_key)
    return Fernet(fernet_key)


def encrypt_openai_key(master_key: bytes, tenant_id: str, plaintext_key: str) -> str:
    """Encrypt tenant's OpenAI API key. Returns ciphertext as string for MongoDB storage."""
    fernet = derive_tenant_fernet_key(master_key, tenant_id)
    encrypted = fernet.encrypt(plaintext_key.encode())
    return encrypted.decode()


def decrypt_openai_key(master_key: bytes, tenant_id: str, encrypted_key: str) -> str:
    """Decrypt tenant's OpenAI API key. Call only in request scope — never persist result."""
    fernet = derive_tenant_fernet_key(master_key, tenant_id)
    return fernet.decrypt(encrypted_key.encode()).decode()


def mask_openai_key(plaintext_key: str) -> str:
    """Return masked key showing last 6 chars: sk-...abc123"""
    if len(plaintext_key) < 6:
        return "sk-...******"
    return f"sk-...{plaintext_key[-6:]}"
```

**Create `app/services/byok_service.py`:**

1. `validate_and_store_openai_key(tenant_id: str, plaintext_key: str) -> str`:
   - Validate key by calling OpenAI's models.list() endpoint (lightweight, no tokens burned):
     ```python
     from openai import AsyncOpenAI, AuthenticationError
     client = AsyncOpenAI(api_key=plaintext_key)
     await client.models.list()
     ```
   - If `AuthenticationError`: raise HTTPException(400, "Invalid OpenAI API key")
   - If other exception: raise HTTPException(503, "Could not validate OpenAI key. Try again later.")
   - Encrypt: `encrypted = encrypt_openai_key(settings.MASTER_ENCRYPTION_KEY.encode(), tenant_id, plaintext_key)`
   - Store in tenant document: `$set: {"encrypted_openai_key": encrypted, "openai_key_last6": plaintext_key[-6:], "updated_at": now}`
   - Return masked key: `mask_openai_key(plaintext_key)`

2. `rotate_openai_key(tenant_id: str, new_plaintext_key: str) -> str`:
   - Same as validate_and_store — validates new key, encrypts, overwrites old (per locked decision: simple overwrite, no version history)
   - This is literally the same function as validate_and_store_openai_key (rotation = store new key)

3. `decrypt_tenant_openai_key(tenant_id: str) -> str`:
   - Fetch tenant document
   - If `encrypted_openai_key` is None: raise HTTPException(402, "OpenAI API key required. Add your key in Settings.")
   - Decrypt: `decrypt_openai_key(settings.MASTER_ENCRYPTION_KEY.encode(), tenant_id, tenant["encrypted_openai_key"])`
   - Return plaintext key (caller uses it in request scope only)

4. `get_openai_key_status(tenant_doc: dict) -> dict`:
   - Return `{"has_key": bool, "masked_key": "sk-...abc123" or None}`
   - Uses `openai_key_last6` field from tenant document for masking (no decryption needed)
