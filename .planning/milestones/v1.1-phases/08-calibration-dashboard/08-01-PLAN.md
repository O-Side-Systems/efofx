**3. Create `app/services/calibration_service.py`:**

```python
CALIBRATION_THRESHOLD = 10  # CALB-03

class CalibrationService:
    async def get_metrics(self, tenant_id: str, date_range: Optional[str] = None) -> dict
```

Implementation details:
- Get tenant-scoped feedback collection via `get_tenant_collection(DB_COLLECTIONS["FEEDBACK"], tenant_id)`
- Build `date_filter` dict: if `date_range == "6months"` → `{"submitted_at": {"$gte": now - 182 days}}`; if `"1year"` → 365 days; if `"all"` or None → empty dict
- Count real outcomes first: `await feedback_col.count_documents(date_filter)`. If < CALIBRATION_THRESHOLD → return `{"below_threshold": True, "outcome_count": count, "threshold": CALIBRATION_THRESHOLD}`
- Build aggregation pipeline with date filter match, then `$lookup` to `reference_classes` using `let`/`pipeline` syntax:
  - `let: {"rc_id": "$reference_class_id", "tenant": tenant_id}`
  - Inner pipeline `$match` with `$expr.$and`: `$eq["$name", "$$rc_id"]` AND (`$eq["$tenant_id", "$$tenant"]` OR `$eq["$tenant_id", None]`) AND `$ne["$data_source", "synthetic"]`
  - CALB-04: explicit tenant_id in inner pipeline — TenantAwareCollection only scopes source
- After `$lookup`, `$unwind` (preserveNullAndEmptyArrays for feedback without matching reference class)
- Group by reference_class_id, compute per-group variance list
- Call `.to_list(None)` on the cursor returned by `await feedback_col.aggregate(pipeline)`
- Compute overall metrics from results using helper functions

Add a second method to CalibrationService:

```python
async def get_trend(self, tenant_id: str, months: int = 12) -> dict
```

Implementation details for `get_trend()`:
- Get tenant-scoped feedback collection via `get_tenant_collection(DB_COLLECTIONS["FEEDBACK"], tenant_id)`
- Count real outcomes first (same threshold check as `get_metrics`). If < CALIBRATION_THRESHOLD → return `{"below_threshold": True, "outcome_count": count, "threshold": CALIBRATION_THRESHOLD, "trend": []}`
- Compute `since = datetime.now(timezone.utc) - timedelta(days=months * 30)` and filter `submitted_at >= since`
- Build an aggregation pipeline that:
  1. `$match` with `submitted_at >= since`
  2. `$project` to compute `variance_pct: { $multiply: [{ $divide: [{ $abs: { $subtract: ["$actual_cost", "$estimate_snapshot.total_cost_p50"] } }, "$actual_cost"] }, 100] }` and `period: { $dateToString: { format: "%Y-%m", date: "$submitted_at" } }`
  3. `$group` by `period`, compute `mean_variance_pct: { $avg: "$variance_pct" }` and `outcome_count: { $sum: 1 }`
  4. `$sort` by `_id: 1` (chronological, oldest first)
  5. `$project` to rename `_id` to `period` and round `mean_variance_pct` to 1 decimal
- Note: This pipeline does NOT need `$lookup` to reference_classes — it operates solely on the feedback collection. The variance is computed from `actual_cost` vs `estimate_snapshot.total_cost_p50` which are both on the feedback document. TenantAwareCollection handles tenant scoping for the source collection.
- Return:
```python
{
    "below_threshold": False,
    "outcome_count": total_count,
    "threshold": CALIBRATION_THRESHOLD,
    "trend": [
        {"period": "2025-10", "mean_variance_pct": 15.2, "outcome_count": 3},
        {"period": "2025-11", "mean_variance_pct": 12.1, "outcome_count": 4},
        {"period": "2025-12", "mean_variance_pct": 9.8, "outcome_count": 5},
    ],
    "months": months,
}
```

Helper functions (module-level, testable independently):
- `_compute_variance(actual_cost: float, estimated_p50: float) -> float` — `abs(actual_cost - estimated_p50) / actual_cost * 100`
- `_compute_accuracy_buckets(variances: list[float]) -> dict` — exclusive slices: [0-10%], (10-20%], (20-30%], >30% as proportions
- `_build_pipeline(date_filter: dict, tenant_id: str) -> list[dict]` — returns the full metrics aggregation pipeline (public for test inspection)
- `_build_trend_pipeline(since: datetime) -> list[dict]` — returns the trend aggregation pipeline (public for test inspection)

Response shape (above threshold):
```python
{
    "below_threshold": False,
    "outcome_count": count,
    "threshold": CALIBRATION_THRESHOLD,
    "mean_variance_pct": round(mean(variances), 1),
    "accuracy_buckets": {
        "within_10_pct": float,  # exclusive [0-10%]
        "within_20_pct": float,  # exclusive (10-20%]
        "within_30_pct": float,  # exclusive (20-30%]
        "beyond_30_pct": float,  # exclusive >30%
    },
    "by_reference_class": [
        {
            "reference_class": str,
            "outcome_count": int,
            "mean_variance_pct": float,
            "accuracy_buckets": {...},
            "limited_data": bool,  # True when outcome_count < 5
        }
    ],
    "date_range": date_range or "all",
}
```

**Variance calculation:** For each feedback doc, compute `abs(doc.actual_cost - doc.estimate_snapshot.total_cost_p50) / doc.actual_cost * 100`. This gives the percentage deviation.

**Important patterns:**
- Motor `aggregate()` returns cursor, not list — use `cursor = await col.aggregate(pipeline)` then `results = await cursor.to_list(None)` (TenantAwareCollection.aggregate returns the cursor from Motor's aggregate, but TenantAwareCollection.aggregate is itself async, so `await` gets the cursor)
- FeedbackDocument.reference_class_id may be None — group as "Unknown" reference class
- Use `datetime.now(timezone.utc)` not `datetime.utcnow()` (deprecated)

```python
from fastapi import APIRouter, Depends, Query
from app.core.security import get_current_tenant
from app.models.tenant import Tenant
from app.services.calibration_service import CalibrationService

calibration_router = APIRouter(prefix="/calibration", tags=["calibration"])

@calibration_router.get("/metrics")
async def get_calibration_metrics(
    tenant: Tenant = Depends(get_current_tenant),
    date_range: str = Query(default="all", pattern="^(6months|1year|all)$"),

    svc = CalibrationService()

@calibration_router.get("/trend")
async def get_calibration_trend(
    tenant: Tenant = Depends(get_current_tenant),
    months: int = Query(default=12, ge=1, le=36),
) -> dict:
    svc = CalibrationService()
    return await svc.get_trend(tenant.tenant_id, months)
```
